## 🚀 快速开始

### 步骤 1: 检查环境

```powershell
# 在项目目录运行测试脚本
.\run_device_test.ps1
```

### 步骤 2: 连接设备

**Android:**
```bash
# 通过 USB 连接 Android 设备
adb devices

# 确保设备已开启开发者模式
# 设置 > 关于手机 > 点击版本号7次
# 设置 > 开发者选项 > 开启 USB 调试
```

**iOS (macOS):**
```bash
# 连接 iPhone/iPad
xcrun simctl list devices available

# 查看已连接设备
flutter devices
```

### 步骤 3: 运行应用

```bash
# 列出所有可用设备
flutter devices

# 运行应用到指定设备
flutter run -d <device_id>

# 例如: flutter run -d android
# 例如: flutter run -d 00001234
```

## 📞 遇到问题?

1. **应用无法启动**: 检查 `flutter doctor`
2. **设备未识别**: 检查 USB 驱动/调试模式
3. **构建失败**: 查看错误信息，运行 `flutter analyze`
4. **崩溃问题**: 查看日志 `flutter logs`
