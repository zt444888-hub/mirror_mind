## 📋 测试说明

### 测试前准备

1. **安装应用**
   - iOS: `flutter run -d <device_id>` 或通过 Xcode 安装
   - Android: `flutter run -d <device_id>` 或通过 ADB 安装

2. **检查设备**
   ```bash
   flutter devices
   adb devices  # Android
   xcrun simctl list devices available  # iOS
   ```

3. **准备测试账号**
   - Pro 功能测试需要准备测试内购
