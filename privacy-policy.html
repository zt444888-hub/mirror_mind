信息收集与存储

您在本应用输入的所有情绪记录仅存储在您的设备本地（SQLite 数据库）。API Key 使用 iOS Keychain 加密存储。本应用不会自动收集任何个人身份信息，也不会将您的任何数据上传至我们控制的任何服务器。

AI 分析

当您使用 AI 分析功能时，您输入的文本通过 HTTPS 加密传输至您自行配置的 API 端点。我们无法访问您的 API Key 或您的情绪记录内容。

数据导出与删除

您可以通过设置页导出数据或清除所有数据。
