## 🚀 上架前必做事项

### iOS App Store 上架流程

1. **准备工作**
   - [ ] 创建 Apple Developer Program 账号（年费 $99/$299）
   - [ ] 创建 App Store Connect 应用
   - [ ] 配置应用名称、描述、截图

2. **构建准备**
   - [ ] 配置 Flutter iOS 发布构建
   ```bash
   flutter build ios --release
   ```
   - [ ] 使用 Xcode 验证构建产物
   - [ ] 生成 iOS App Store 包

3. **提交审核**
   - [ ] 上传构建产物到 App Store Connect
   - [ ] 选择构建版本
   - [ ] 填写应用信息
   - [ ] 提交审核

4. **审核清单**
   - [ ] 隐私政策已部署且可访问
   - [ ] 所有权限有明确用途说明
   - [ ] 应用内购测试通过
   - [ ] 无崩溃、无明显Bug

### Android Google Play 上架流程

1. **准备工作**
   - [ ] 创建 Google Play Console 账号（注册费 $25）
   - [ ] 创建应用
   - [ ] 填写商店信息

2. **构建准备**
   - [ ] 配置 release 签名
   - [ ] 构建 AAB 包
   ```bash
   flutter build appbundle --release
   ```
   - [ ] 上传 AAB 到 Play Console

3. **发布配置**
   - [ ] 配置国家/地区发布范围
   - [ ] 配置价格（免费/付费）
   - [ ] 配置内容分级

4. **提交审核**
   - [ ] 上传 AAB
   - [ ] 填写应用信息
   - [ ] 提交审核
