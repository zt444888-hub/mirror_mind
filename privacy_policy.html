二、AI 分析的数据处理

心镜的情绪分析功能调用外部 AI API。为保护您的隐私，我们采取以下措施：

  1. 仅发送情绪文本：仅将您输入的情绪描述文本（不含任何个人身份标识）发送至 AI API。
  2. 不传输身份信息：API 请求中不包含用户 ID、设备 ID 或任何可关联至具体个人的标识符。
  3. 数据脱敏：在发送前对文本进行脱敏处理，移除可能含有的个人识别信息（如人名、地名、电话号码等格式的文本会自动过滤）。
  4. 不留存：AI API 的请求和响应不会被我们存储或记录，仅将分析结果保存至您的本地设备。

三、数据安全

3.1 存储安全

  • 所有情绪记录、日记内容和分析结果存储在设备本地的 SQLite 数据库中。
  • 数据库使用 AES 加密算法进行加密存储，防止未授权访问。
  • 应用沙盒机制确保其他应用无法访问心镜的数据文件。

3.2 数据传输

  • 心镜不向自有服务器上传任何用户数据。
  • AI API 通信使用 HTTPS 加密传输。
  • 应用内购买交易通过 Apple / Google 官方支付渠道完成，心镜不处理您的支付信息。
