隐私政策

更新日期：2026年6月

应用名称：心镜 - 情绪日记

1. 信息收集

本应用不会收集您的个人身份信息（如姓名、地址、电话号码等）。

您在使用过程中输入的情绪记录、日记内容仅保存在您的设备本地。

2. AI 分析功能

当您使用 AI 情绪分析功能时，您输入的文本内容会被发送到云端服务器进行处理。

分析完成后，服务器不会保留您的文本内容。

您配置的 API Key 仅保存在设备本地，通过系统安全存储。

3. 第三方服务

本应用使用了以下第三方服务：

- AI 分析服务（用于情绪分析）

- 应用内购买（由 Apple 提供）

4. 数据安全

我们采用合理的措施保护您的数据安全。

所有本地存储的数据受 iOS 系统沙箱保护。

5. 用户权利

您可以随时：

- 在应用设置中删除所有本地数据

- 清除所有情绪记录

- 关闭 AI 分析功能

本隐私政策可能会不时更新。更新后的政策会在应用内或本页面公布。
